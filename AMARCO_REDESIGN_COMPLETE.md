## Design System

### Color Palette
```
Primary Gold: #D4AF37
Light Gold: #E8C968
Dark Gold: #B8941F

Backgrounds:
- Black: #0A0A0A
- Dark Gray: #1A1A1A
- Medium Gray: #2A2A2A

Text:
- Primary: #FFFFFF
- Secondary: #B0B0B0
- Tertiary: #808080

Glass Effects:
- Light: #30FFFFFF (19%)
- Medium: #20FFFFFF (12%)
- Dark: #10FFFFFF (6%)
```

### Typography Scale
```
Display: 48sp / 36sp
Headline: 32sp / 24sp / 20sp
Title: 18sp / 16sp / 14sp
Body: 16sp / 14sp / 12sp
Label: 14sp / 12sp / 10sp (uppercase)
```

### Focus States
```
Normal: 1dp subtle border
Focused: 3dp gold border + outer glow
Pressed: Darker background
Transitions: 200-300ms smooth
```
